# Studentské miniprojekty

V tomto repositáři najdete pro inspiraci řešení mnoha malých žákovských projektů. Níže naleznete přehled jednotlivých projektů a jejich zaměření.

## Přehled projektů

- **[ctenost-knih-webova-aplikace](./ctenost-knih-webova-aplikace/)**
  Webová aplikace (HTML, CSS, JS, PHP) pro sběr a zobrazení statistik o přečtených knihách od návštěvníků. Obsahuje formulář, uložení dat a zobrazení přehledové tabulky.

- **[filmova-databaze](./filmova-databaze/)**
  Návrh a implementace rozsáhlejší filmové databáze (představující entity filmy, osoby a žánry). Projekt pokrývá použití technologií jako SQL, HTML, SASS, JS, PHP, OOP, ORM a MVC.

- **[flights](./flights/)**
  Vizualizace odletové tabule na letišti. Frontend pomocí JavaScriptu a AJAXu pravidelně stahuje data přes PHP backend napojený na databázi MySQL a aktualizuje odletovou desku v reálném čase.

- **[generovany-web](./generovany-web/)**
  Ukázkový postup přechodu z čistě statického HTML webu na dynamicky generovaný web (rozklad na šablony a znovupoužití částí stránky).

- **[kryptografie](./kryptografie/)**
  Komplexní úloha zaměřená na kryptografii, která se skládá ze tří částí: 
  1. Edukační responzivní web v HTML/CSS/JS ukazující základní šifry,
  2. Architektura databáze pro evidenci šifrovaných zpráv a pokusů o rozluštění (SQL),
  3. Skripty v PHP/Pythonu simulující použití transpoziční šifrovací mřížky.

- **[monty-hall-problem](./monty-hall-problem/)**
  Čistě JavaScriptový simulační skript slavného pravděpodobnostního paradoxu *Monty Hall Problem*. Skript opakovaně spouští volby dveří a ilustruje úspěšnost (win-rate).

- **[simple-chat](./simple-chat/)**
  Jednoduchá live webová chatovací aplikace využívající backend psaný v Node.js/Express.

- **[vericheck](./vericheck/)**
  Vzdělávací nástroj ve formě rozšíření (extension) pro webové prohlížeče (Chrome/Edge/Brave). Slouží k prevenci i varování před dezinformacemi, clickbaity a identifikaci manipulativního obsahu na stránkách pomocí analýzy DOMu a detekce podezřelých URL.
